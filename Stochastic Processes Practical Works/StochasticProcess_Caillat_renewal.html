def spoi(lam,T):
    sauts = [0]
    C = (sauts[-1]<T)
    while(C):
        t = sauts[-1] + rnd.exponential(scale=1/lam)
        if (t<T):
            sauts.append(t)
        else:
            C = False
    return(sauts)

def sspoi(lam, T):
    # N ~ P(T)
    N = rnd.poisson(lam*T)
    # N uniformes sur [0,T]
    U = (rnd.uniform(0, T, N))
    # Trie les N uniformes
    U.sort()
    return(U)

Taille optimale d'un buffer¶

On fixe les valeurs du cot de vidage $K=100$ euros et $\lambda=1$, i.e. on a un message en moyenne par unit de temps. En faisant varier le cot unitaire $h$ de stockage d'un message (valeurs $0.1,1,10 $ ou $100$), calculer le cot moyen par cycle en effectuant un grand nombre (n=500) de cycles, et ce pour suffisamment de valeurs de $T$ pour identifier la valeur optimale.

On pourra crire d'abord une fonction

$\tau_{t} = \frac{1}{T} (h {\sum \limits_{i=1}^{N(t)} (T - t_{i})} + K)$ $\tau_{t}$ est le cot unitaire observ pour la ralisation d'un cycle.

In [15]:
def coutcycle(T,h,K,lam):
    A = sspoi(lam, T)
    diff = []
    for i in range(len(A)):
        diff.append(T-A[i])
    tau = (h*sum(diff) + K)/T
    return(tau)

def coutmoyemp(T,h,K,lam,n):
    X = 0
    for i in range(n):
        X += coutcycle(T,h,K,lam)
    X = (1/n)*X
    return(X)
   
n= 500
c_moy = coutmoyemp(100, h, K, lam, n)
