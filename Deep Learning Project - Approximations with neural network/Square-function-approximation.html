Sawtooth method implementation¶

In [4]:
def f(m):
    fm = Sequential()
    
    fm.add(Dense(4, input_dim=1,
                 kernel_initializer="ones",
                 bias_initializer=initializers.constant([0, -1/2, -1, 0])))
    fm.add(Activation("relu"))
    
    for s in range(1, m):
        fm.add(Dense(4, 
                     kernel_initializer=initializers.constant([[ 2 ,  2 ,  2 , -2/2**(2*s) ],
                                                               [-4 , -4 , -4 ,  4/2**(2*s) ],
                                                               [ 2 ,  2 ,  2 , -2/2**(2*s) ],
                                                               [ 0 ,  0 ,  0 ,           1 ]]),
                     bias_initializer=initializers.constant([0, -1/2, -1, 0])))
        fm.add(Activation("relu"))
    
    fm.add(Dense(1, kernel_initializer= initializers.constant([-2/2**(2*m), 4/2**(2*m), -2/2**(2*m), 1])))
    fm.add(Activation("relu"))
    
    
    return fm

Parameters enumeration¶

We are looking to assess the approximation error vs. the number of parameters. Therefore, the parameters number shall be calculated as a function of $m$.

In [8]:
# input: neural network
# output: number of weights of the neural network

def norm0(nn):
    W = nn.get_weights()
    r = 0
    for w in W:
        p = 1
        for s in w.shape:
            p *= s
        r += p
    
    return r

Optimization attempt¶

The approximation $f(m)$ is an raw interpolation of the target function. For it is not the best approximation assessable, the model shall be optimized using a gradient descent process on the neural network parameters, starting from $f(m)$.

In [12]:
err_opt = []

# Error estimation set of points
x_test = np.linspace(dm, dM, Ntest)
y_test = target(x_test)

# Optimization set of points
Ntrain = 10000
x_train = np.random.uniform(dm, dM, Ntrain)
y_train = target(x_train)

# Optimization calculation for each m
for m in mGrid:
    # Approximation using sawtooth method
    fm = f(m)
    
    # Model optimization
    fm.compile(optimizer = optimizers.Adam(err_approx[m-1]/10), loss = losses.mean_absolute_error)
    fm.fit(x_train, y_train, epochs=3, batch_size=100, verbose=0)
    fm.compile(optimizer = optimizers.Adadelta(err_approx[m-1]/10), loss = losses.mean_absolute_error)
    fm.fit(x_train, y_train, epochs=3, batch_size=100, verbose=0)
    
    # Error from optimized model estimation
    y_pred = fm.predict(x_test)
    err_opt.append(np.mean(np.abs(y_pred.reshape(1, -1) - y_test)))
err_opt = np.array(err_opt)

# Display of the different errors 
plt.plot(weights_fm, np.log(err_approx), label="log(err)")
plt.plot(weights_fm, np.log(err_opt), label="log(err_opt)")
plt.legend()
plt.xlabel("complexity")
plt.show()
